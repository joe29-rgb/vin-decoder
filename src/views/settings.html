üìö How Black Book Integration Works
With API Key: The system will automatically fetch Black Book values for your inventory when you scrape or upload vehicles.

Without API Key: You can manually enter Black Book values using the "Set BB" button in the inventory table.

Default Behavior: When Black Book value is missing or zero, the system uses $10,000 as a default for deal calculations.

Getting API Access: Visit canadianblackbook.com to sign up for API access.
